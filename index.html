Instructions

This page displays all cards (Global Actions not being counted) available in Dukhlon Tales. The cards can be filtered to only show cards you are interested in. Multiple filters in the same column use OR logic (they will display cards that fit any criteria). Multiple filters in different columns use AND logic (they will display cards that fit every criteria).

When filtering for cards that can be used with certain weapons, select all filters that match the weapons' characteristics (such as Attack type, melee/ranged categorization, and handedness). The "Any Weapon" filter displays cards that require a weapon, but can be used with any weapon. The "Non Weapon" filter displays cards that don't require weapons.
